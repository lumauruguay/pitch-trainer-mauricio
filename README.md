# 🎯 BlueIA Pitch Trainer — Mauricio (Imperio del Este)

Entrenador de pitch interactivo con voz para la visita a **Mauricio**, dueño del Supermercado El Imperio (Salinas, Canelones, Uruguay).

## 🚀 Cómo usar

**Sin instalación. Abrí directamente en tu browser:**

```
https://lumauruguay.github.io/pitch-trainer-mauricio/
```

O cloná el repo y abrí `index.html` localmente.

## 🎙️ Funcionalidades

- **5 personalidades de Mauricio**: Curioso, Ocupado, Escéptico, Economista, Abierto
- **Árbol de decisiones** NAME → SAME → FRAME → AIM → GAME (metodología Priestley)
- **Voz bidireccional**: Mauricio te habla (Web Speech Synthesis) + vos respondés por micrófono (Web Speech Recognition)
- **Feedback inmediato** con citas de Priestley y Meléndez
- **Score adaptativo**: streak, puntos por calidad de respuesta
- **Cheatsheet flotante** con el guion completo siempre a mano
- **Modo oscuro/claro** automático
- **100% gratis** — sin API keys, sin backend, funciona offline
